## 故障屏蔽规则
1. **手动屏蔽**
    1. 规则：页面配置屏蔽规则（分南向和北向）
2. **检修屏蔽**
    1. 规则：电梯处于检修模式状态
3. **停用状态屏蔽**
    1. 规则：设备处于停用状态
4. **传感器关联配置状态屏蔽**
    1. 规则：设备传感器配置的参数状态会影响关联的故障上报
5. **传感器关联故障屏蔽**
    1. 规则：设备上报的传感器故障大于2次，会屏蔽该传感器对应的故障
6. **故障上报次数限制屏蔽**
    1. 规则：南向屏蔽（平台）每小时故障数 > 3
   北向（第三方）每天推送不超过5次
